# Media folder

Put your files here. Names must match what's referenced in `../js/data.js`.

- `img/profile.jpg` — your headshot (About section)
- `thumbs/*.jpg` — project card cover images (16:10 ratio looks best)
- `video/*.mp4` — short 20–30 s demo clips (H.264 MP4, keep under ~25 MB each)
- `Vinith-Kumar-Resume.pdf` — your CV (enables the Download CV button)

Anything missing just shows a 🤖 placeholder — nothing breaks.
