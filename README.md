# Vinith Kumar T — Portfolio

Live site → **https://vinithkumar-themorgan.github.io**

Personal portfolio of Vinith Kumar T — Full Stack Robotics Engineer
(drones, AMRs, six-axis robots, RF & waveguide, computer vision).

## Edit your content

Almost everything lives in **`assets/js/data.js`** — skills, projects,
experience, achievements. Just edit the text there.

## Update the live site

```bash
git add -A
git commit -m "update"
git push
```

It goes live in about a minute.

## Preview locally

Open `index.html` in a browser, or run:

```bash
python -m http.server 8000
```

then visit http://localhost:8000

## Media

- Images → `assets/media/img/` and `assets/media/thumbs/`
- Videos → `assets/media/video/`

Use **lowercase filenames** (GitHub is case-sensitive).
